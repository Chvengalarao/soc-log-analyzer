# SOC Log Analyzer

A Python-based cybersecurity tool that scans security logs and detects suspicious activity.

## Features
- Detects brute force login attempts
- Detects port scanning activity
- Detects malware indicators
- Extracts attacker IP addresses
- Generates a SOC investigation report

## Files
soc_analyzer.py → Python detection engine  
logs.txt → Sample security logs for testing

## How to Run

python soc_analyzer.py

## Use Case

This project simulates basic Security Operations Center (SOC) log analysis and incident detection.
